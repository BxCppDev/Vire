====================================
Vire CMS example ex01
====================================



Preparation
-----------

Setup Vire and its dependees from a Cadfaelbrew shell:

.. code:: sh

	  $ brew sh
	  $ bayeux_dev_setup
	  $ vire_dev_setup
	  $ which virequery
	  ...
..

Build
-----------

From the ``examples/cms/ex01`` source directory:

.. code:: sh

	  $ ./tools/build.sh
	  ...
..
